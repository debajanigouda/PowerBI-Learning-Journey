# Day 01 — Power Query Basics

## What I Learned
- Fill: fills empty cells downward with value above
- FillDown: same as Fill, applied downward
- Pivot: converts row values into columns
- Unpivot: converts columns back into rows
- Rename: rename column headers cleanly
- Move: reorder columns left or right
- Replace Errors: fix #ERROR values in data
- Data Types: set correct type — text, number, date

## Dataset Used
- StudentData.csv — student names + activities
- StudentTestingData.csv — 2,000 student records
